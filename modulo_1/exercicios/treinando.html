HTML

oque é ?

HTML é a linguagem padrao para criar páginas da web. Ela funciona como a "estrutura/esqueleto" de um sit, definindo onde ficam textos, imagems, links, videos e outros elementos

Composicao

um documento HTMl é composto por tags(etiquetas), que informam ao navegador como cada parte do conteudo deve ser exibida. Por exemplo:

  • <h1> cria titulos,
  • <p> cria parágrafos,
  • <a> cria links,
  • <img> cria imagems.

estrutura base

    

<!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
    </head>
    <body>
        
    </body>
    </html>

como acontece ?

quando voce abre esse arquivo no navegador, ele interpreta o código e mostra o contéudo de forma organizada.

resumo

O HTML é o esqueleto da web, simples, mas essencial! Quer deixar bonito? Ai entra as CSS. Quer interatividade ? Ai entra o JavaScript.
